*# TP6 - Réveil Géographique

Résumé
- Projet pédagogique (Licence 1) : implémentation d'un TP AppInventor2 / page web présentant les notions de géolocalisation, distance orthodromique, cap (bearing) et système d'alerte.
- Fichiers principaux : [index.html](index.html).

Fonctionnalités
- Présentation des formules de distance et du mille marin.
- Pseudo‑codes AppInventor2 et équivalents JavaScript.
- Interface web (voir [index.html](index.html)) permettant d'ajouter des captures d'écran via les helpers JavaScript : [`handleScreenshots`](index.html), [`createScreenshotItem`](index.html) et [`openModal`](index.html).

Installation / utilisation locale
1. Cloner ou copier le dépôt sur votre machine.
2. Ouvrir le fichier [index.html](index.html) dans un navigateur web (double‑clic ou via "Open File" dans VS Code).
